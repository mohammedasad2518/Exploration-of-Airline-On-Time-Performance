## 🛫 Overview

**Exploration of Airline On-Time Performance** is a data analytics and exploratory data analysis (EDA) project focused on U.S. domestic airline operations.  
The notebook analyzes flight delay data to discover patterns in airline punctuality, arrival/departure delays, and state-wise delay distributions.

The project uses real-world aviation datasets and visual analytics techniques to transform raw flight records into meaningful insights.

---

## 📌 Objectives

This project aims to answer the following questions:

- What is the distribution of departure delays greater than 15 minutes by state?
- How are arrival delays distributed across the country?
- Do flights between certain states experience more delays?
- How did arrival delays vary day-by-day during June 2014?


